# kataflow

Usage:
```
nodejs kata.js
```

**kata.js** contains two functions:

- *add* : the core function
- *test* : prints input and output on console

Some tests are included in the file:
```
test("");
test("5");
test("2,4");
test("1,2,3");
test("1,-2,-3");
test("1,4,7,10,");
test("1,1005,3");
test("2,1000,3");
test("4,5\n6");
test("//;\n1;3;5");
test("//\n\n1\n3\n5");
```
